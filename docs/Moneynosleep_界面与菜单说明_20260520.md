# Moneynosleep 界面与菜单说明

日期：2026-05-20

## 1. 当前界面结构

当前 Streamlit 界面由两部分组成：

- 主工作台：`streamlit app`
- 说明页菜单：位于左侧导航栏

主工作台负责真实操作，说明页负责解释“这是什么、怎么用”。

## 2. 左侧菜单说明

### streamlit app

主工作台。

主要功能都在这里完成，包括：

- miniQMT 连接测试
- miniQMT / BaoStock 行情同步
- 日线快速复盘
- 5分钟回踩复盘
- 候选股查看
- 信号查看
- 交易列表查看
- K线复核
- 人工验证
- 问题归因统计

### 使用说明

面向使用者的总入口。

主要说明：

- 这些菜单分别是什么
- 推荐使用顺序
- 一键启动方式
- 当前版本边界

### 数据同步

说明如何把外部行情拉进本地数据库。

涉及：

- miniQMT
- BaoStock
- AKShare
- DuckDB
- Parquet

### 条件选股

说明当前日线筛选逻辑。

当前重点条件：

- `close > ma55`
- `volume_ratio_5 > 阈值`

### 连续复盘

说明当前两类复盘：

- 日线快速复盘
- 5分钟回踩复盘

### 交易复核

说明如何：

- 看交易结果
- 看 K 线买卖点
- 做人工复核
- 填写问题标签和备注

### 问题归因

说明复核后的问题统计用途。

主要用来：

- 找重复错误
- 统计执行问题
- 区分策略问题与交易问题

### 自选观察

预留功能页。

后续会承载：

- 自选股池
- 重点观察标的
- 复盘后跟踪清单

## 3. 推荐使用顺序

1. 双击 `启动Moneynosleep.cmd`
2. 打开 `streamlit app`
3. 在侧边栏做 `miniQMT 连接测试`
4. 做 `miniQMT 行情同步`
5. 运行 `快速复盘` 或 `5分钟回踩复盘`
6. 查看 `候选与信号`
7. 查看 `交易列表`
8. 在 `K线复核` / `人工验证` 中做人工检查
9. 到 `问题归因` 查看汇总

## 4. 当前版本边界

当前版本重点是：

- 数据同步
- 本地存储
- 日线筛选
- 快速复盘
- 5分钟回踩复盘
- 人工复核

当前还没有开放：

- 真实自动下单
- 实盘交易风控联动
- 独立自选池工作流
